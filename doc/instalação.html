    Guia de Instalação

    Dependências:

    1. hostapd: Assistente para access point e servidores de autentificação. Usado para criar um ponto de acesso sem fio utilizando Linux.

    2. dhcpd (isc-dhcp-server): Usado como servidor de DHCP.

    Para Instalar as Dependências, execute:

    $ sudo apt-get install hostapd
$ sudo apt-get install isc-dhcp-server

    Após ter instalado as dependências acima, basta executar o programa que ele configurará automaticamente o seu sistema.

    Atenção! Todas as configurações de rede serão substituidas para que o computador funcione exclusivamente como um access point.
